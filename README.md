# Shooter Data Manager (SDM) for Windows

Welcome to the Shooter Data Manager (SDM) for Windows, developed by Redmer Controls Inc. This application is an order entry platform for the Lucht Shooter System, the industry standard production system for professional photographic labs (Pro-Labs) from 1985 through 2010. This application replaced a DOS based application written by Ronald Redmer in 1985 and provided the following benefits: 

- Support for Microsoft Windows
- Ability to edit multiple files simultaneously
- Ability to copy and paste between files

## Introduction

Shooter Data Manager (SDM) is a comprehensive tool for managing shooter data. It allows users to handle various data files, perform edits, and generate reports. The application is built using Visual Basic and is designed to run on Windows systems.

## Features

- **File Management**: Easily manage and edit shooter data files.
- **Report Generation**: Generate detailed reports from the data.
- **User Interface**: Intuitive and user-friendly interface.
- **Error Handling**: Robust error handling to ensure smooth operation.
- **Compatibility**: Compatible with various Windows versions.

## Installation

To install SDM for Windows, follow these steps:

1. Download the latest release from the [releases page](https://github.com/your-repo/museum-redmer-controls-sdm/releases).
2. Extract the downloaded files to your desired location.
3. Run `setup.exe` to start the installation process.
4. Follow the on-screen instructions to complete the installation.

## Usage

After installation, you can start using SDM by running `SDM.exe`. The main features of the application include:

- **File Manager**: Accessed via the `FileManager` form, allows you to select and edit SDM files.
- **Report Form**: Accessed via the `ReportForm` form, displays report information.
- **Log Viewer**: Accessed via the `ViewLog` form, provides application tracking and timing routines.

### Example

To load and edit a file:

1. Open SDM.
2. Navigate to the `FileManager` form.
3. Select the file you want to edit.
4. Make the necessary changes and save the file.
